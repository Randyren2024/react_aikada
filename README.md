# AIKada 前端应用

## 项目概述
这是一个基于React和Vite的现代化前端应用，集成了AIKada后端API服务，提供用户认证、活动管理、打卡记录、社交互动和群组功能。

## 技术栈
- React 18
- Vite
- Tailwind CSS
- React Hooks
- Fetch API

## 快速开始

### 前置条件
- Node.js 16+
- npm 或 yarn
- AIKada后端服务已启动 (http://localhost:5000)

### 安装依赖
```bash
cd frontend
npm install
```

### 启动开发服务器
```bash
npm run dev
```

应用将在 http://localhost:5174 启动（如果5173端口被占用）

### 构建生产版本
```bash
npm run build
```

## 功能特性

### 用户认证
- 用户注册和登录
- JWT令牌管理
- 自动登录状态维护

### 活动管理
- 浏览推荐活动
- 创建新活动
- 活动详情查看

### 打卡记录
- 创建打卡记录
- 查看个人打卡历史
- 打卡统计展示

### 社交互动
- 发布动态帖子
- 浏览社区动态
- 用户互动功能

### 群组功能
- 创建学习群组
- 浏览推荐群组
- 加入/退出群组

## API集成

前端应用通过 `src/api/index.js` 模块与后端API进行通信：

- **认证API**: 用户注册、登录、令牌管理
- **活动API**: 活动CRUD操作
- **打卡API**: 打卡记录管理
- **社交API**: 帖子发布和管理
- **群组API**: 群组操作和管理

## 错误处理

应用实现了完善的错误处理机制：
- 网络连接失败提示
- 401认证失败处理
- 用户友好的错误信息展示

## 项目结构

```
src/
├── api/              # API调用模块
│   └── index.js     # 所有API接口定义
├── components/       # React组件
│   └── Login.jsx    # 登录/注册组件
├── AppWithAPI.jsx    # 主应用组件
├── main.jsx         # 应用入口
└── index.css        # 全局样式
```

## 注意事项

1. 确保后端服务已启动并运行在 http://localhost:5000
2. 如果遇到网络连接错误，请检查后端服务状态
3. 开发模式下使用5174端口，如需要可修改配置
4. 所有API调用都包含错误处理和用户反馈

## 部署

构建后的文件位于 `dist/` 目录，可以部署到任何静态文件服务器。
